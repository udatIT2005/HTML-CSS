# Responsive web design là gì ?

    <meta charset="UTF-8" />
    - Mã hóa ký tự 
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    - Render dúng tỷ lệ theo màn hình thiết bị 
    <title>Document</title>
    - Tên website
